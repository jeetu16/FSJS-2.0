H1 is block level element

span is inline element This is another span element but it is converted into block element

Inline element
We can not increase the height and width of Inline Element.It always taken the content width.
Block element
Block element always take full width of the parent element. We can increase or decrease the height and width of Block Element. Block element always starts with new line
Inline-Block
Inline-Block same as block element only difference is its always as per content width but we can increase or decrease the width and height of Inline-Block
